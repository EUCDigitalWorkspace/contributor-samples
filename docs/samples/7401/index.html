Create App Volumes vCenter Roles

Download from GitHub.

This script is used to create a new roles on your vCenter server for App Volumes 2.x, 3.x and 4.x     The newly created role will be filled with the needed permissions for App Volumes     The permissions are based on the documentation found here: https://docs.vmware.com/en/VMware-App-Volumes/4/com.vmware.appvolumes.admin.doc/GUID-505624F3-F3EB-428C-BEA0-5BD7F6095A1F.html
